# OCTAVE Allegro Overview

OCTAVE Allegro is an asset-centric information security risk assessment methodology.

Core phases:

1. Establish Risk Measurement Criteria
2. Develop Information Asset Profiles
3. Identify Asset Containers
4. Identify Areas of Concern
5. Develop Threat Scenarios
6. Identify Risks
7. Analyze Risks
8. Select Mitigation Approaches

This repository demonstrates the methodology and practices applied during an enterprise cybersecurity risk assessment engagement.
